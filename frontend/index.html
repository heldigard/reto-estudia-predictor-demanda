Cómo usar esta entrega

Modo live

Si defines window.APP_CONFIG.apiBaseUrl o agregas ?apiBaseUrl=https://tu-url, el frontend consulta el backend FastAPI.

Se descartó como backend principal porque exige túneles efímeros y una sesión viva. La opción estable incluida es backend local más port forwarding.
